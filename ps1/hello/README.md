# Hello, me

Implement a program that prompts the user for their name and then says hello to that user.
For instance, if the user’s name is Adele, your program should print `hello, Adele\n`!

```bash
$ ./hello
What's your name? Adele
hello, Adele
$ ./hello
What's your name?
```

Recall that you can get a string from a user with `getString`.
Recall that you can print a string with `printf`, which is declared in `stdio`.
Recall that you can format a string with `printf` with `%s`.
